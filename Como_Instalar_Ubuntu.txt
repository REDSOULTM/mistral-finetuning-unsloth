# 🚀 Cómo instalar Ubuntu (WSL2) en Windows para usar este proyecto

---

## ¿Para qué sirve esto?

Si tenés Windows y querés entrenar tu propio modelo Mistral-7B usando tu GPU, esta guía te explica cómo instalar Ubuntu (WSL2), dejar todo listo y correr el proyecto sin vueltas. Es paso a paso, para que no te pierdas.

---

## 1) Instalar Ubuntu 22.04 en WSL2

- Abrí **PowerShell como administrador** (buscá "PowerShell", clic derecho, "Ejecutar como admin")
- Pegá esto y dale Enter:
  ```powershell
  wsl --install -d Ubuntu-22.04
  ```
- Esperá que termine y reiniciá si te lo pide
- Abrí la app "Ubuntu 22.04" desde el menú Inicio
- Elegí usuario y contraseña de Linux (anotalos, los vas a usar)
- Verificá que está todo bien:
  ```powershell
  wsl --version
  wsl --list --verbose
  ```

---

## 2) Instalar driver NVIDIA (soporte WSL)

- En Windows, bajá el **driver NVIDIA** más nuevo desde la web oficial
- Instalalo y reiniciá
- Abrí Ubuntu y probá:
  ```bash
  nvidia-smi
  ```
- Si ves tu GPU, ¡listo! Si no, reinstalá el driver

---

## 3) Preparar Ubuntu

- Actualizá todo y poné lo básico:
  ```bash
  sudo apt update && sudo apt upgrade -y
  sudo apt install -y python3 python3-venv python3-pip git build-essential curl wget
  ```
- Probá que Python esté bien:
  ```bash
  python3 --version
  ```

---

## 4) Instalar VS Code y extensiones

- Bajá **VS Code** en Windows
- Instalá estas extensiones en VS Code:
  - Remote - WSL
  - Python
  - Jupyter
- Abrí una ventana remota: `Ctrl + Shift + P` → "Remote-WSL: New Window"

---

## 5) Descargar el proyecto de GitHub

- En Ubuntu:
  ```bash
  mkdir -p ~/projects
  cd ~/projects
  git clone https://github.com/REDSOULTM/mistral-finetuning-unsloth.git
  cd mistral-finetuning-unsloth
  ls -la
  ```
- Si ya lo bajaste en Windows, podés copiarlo a Ubuntu:
  ```bash
  cp -r "/mnt/c/Users/TuUsuario/Downloads/mistral-finetuning-unsloth" ~/projects/
  cd ~/projects/mistral-finetuning-unsloth
  ```

---

## 6) Crear el entorno de Python

- En la carpeta del proyecto:
  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
  python --version
  pip install --upgrade pip
  ```
- Cada vez que abras Ubuntu, activá el entorno:
  ```bash
  cd ~/projects/mistral-finetuning-unsloth
  source .venv/bin/activate
  ```

---

## 7) Instalar dependencias (la forma fácil)

- Usá el instalador automático:
  ```bash
  cd "Instalar Requesitos"
  python InstalarTodo.py
  ```
- Si algo falla, instalá manual:
  ```bash
  pip install torch --index-url https://download.pytorch.org/whl/cu121
  pip install transformers accelerate datasets peft sentencepiece bitsandbytes
  pip install "trl<0.9.0" ipykernel jupyter
  pip install "unsloth[colab-new] @ git+https://github.com/unslothai/unsloth.git"
  ```
- Verificá que todo esté OK:
  ```bash
  cd ..
  cd "Instalar Requesitos"
  python VerificarTodo.py
  ```

---

## 8) Abrir el proyecto en VS Code y entrenar

- En Ubuntu:
  ```bash
  cd ~/projects/mistral-finetuning-unsloth
  code .
  ```
- En VS Code, elegí el intérprete de Python: `Ctrl + Shift + P` → "Python: Select Interpreter" → `./.venv/bin/python`
- Para entrenar:
  ```bash
  cd "RealizarFineTuning"
  python mistral_finetuning_final.py
  ```
- Elegí la opción de dataset que más te sirva

---

## 9) Tips y problemas comunes

- **Activar entorno:**
  ```bash
  cd ~/projects/mistral-finetuning-unsloth
  source .venv/bin/activate
  ```
- **Ver uso de GPU:**
  ```bash
  nvidia-smi
  ```
- **No aparece Python en VS Code:**
  - `Ctrl + Shift + P` → "Python: Select Interpreter" → `./.venv/bin/python`
- **CUDA out of memory:**
  - Cambiá el batch size en `mistral_finetuning_final.py` a 1
- **No encuentro mis archivos:**
  - Ubuntu: `~/projects/mistral-finetuning-unsloth/`
  - Windows: `\\wsl$\Ubuntu-22.04\home\TuUsuario\projects\mistral-finetuning-unsloth\`
